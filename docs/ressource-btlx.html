BTLx Parser

Une bibliothèque Python légère et sans dépendance qui transforme les fichiers BTLx — le format d'échange ouvert des machines de taille CNC (Hundegger & co.) — en données exploitables : pièces, usinages, listes de débit, volumes et poids. Pour un humain comme pour un agent.

Voir sur GitHub → Documentation
pip install btlx-parser Python 3.8+ 0 dépendance ~99% testé XXE-hardened

En bref

Liste de débit

Regroupe les pièces identiques, calcule longueurs, volumes (m³) et poids (kg).

Usinages typés

Coupes, perçages, tenons, mortaises… classés par catégorie, avec leurs paramètres.

Exports prêts

JSON structuré, CSV (débit & pièces), ou dictionnaire Python immuable.

CLI & lib

Outil en ligne de commande btlx et API Python typée (py.typed).

Prise en main

# Installation
pip install btlx-parser

# Ligne de commande
btlx chantier.btlx              # synthèse lisible
btlx chantier.btlx --cutlist    # liste de débit CSV
btlx chantier.btlx --json       # données structurées

# En Python
import btlx
doc = btlx.parse_file("chantier.btlx")
print(doc.project.name, doc.part_count)
print(btlx.cutlist_csv(doc))

Français — pour qui ?

Charpentiers, bureaux d'études et éditeurs de logiciels CAO/FAO bois qui veulent lire un BTLx sans réécrire un parseur XML à chaque projet : métré, export tableur, contrôle qualité, ou alimentation d'un agent IA.

  • Indépendant de la version (BTLx 1.x / 2.x) et des espaces de noms
  • Modèle immuable + attributs bruts conservés (zéro perte)
  • Position des pièces (X/Y/Z) pour l'assemblage
  • Durci contre les attaques XXE / « billion laughs »

English — what it does

Turns BTLx files (the open XML exchange format of CNC carpentry machines) into clean, typed data — parts, processings, cut lists, volumes and weights — equally good for a spreadsheet export, a cost estimate, or feeding a BTLx file to an LLM agent.

  • Namespace- and version-agnostic (BTLx 1.x / 2.x)
  • Immutable model, raw attributes preserved (no data loss)
  • Part positions (X/Y/Z) for assembly placement
  • Hardened against XXE / billion-laughs attacks

Deutsch — wofür?

Liest BTLx-Dateien (das offene XML-Austauschformat der CNC-Abbundmaschinen) und wandelt sie in strukturierte Daten um: Bauteile, Bearbeitungen, Stücklisten, Volumen und Gewichte — für Tabellen-Export, Kalkulation oder die Übergabe an einen KI-Agenten.

  • Namespace- und versionsunabhängig (BTLx 1.x / 2.x)
  • Unveränderliches Modell, Rohattribute erhalten
  • Bauteilposition (X/Y/Z) für die Montage
  • Gehärtet gegen XXE / Billion-Laughs-Angriffe
